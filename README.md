BiggBoss
========

This is a simple tool to alert about new biggboss tv episodes - a show by
Colors Entertainment  

###Usage:

To check for latest episode:  
``` ./biggboss-checker.py   ```  

To see the man page:  
``` ./biggboss-checker.py --help ```  
``` ./biggboss-checker.py --verbose ```
